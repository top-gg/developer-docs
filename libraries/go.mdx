The Top.gg Go SDK is a community-maintained library that provides access to the Top.gg REST API from Go applications. It is not maintained by the Top.gg core team. For installation instructions, import paths, and usage examples, refer directly to the GitHub repository.

## Getting started

**[github.com/top-gg/go-sdk](https://github.com/top-gg/go-sdk)**

<Note>
  The repository README contains the canonical `go get` command, module path, and code examples. Always check there for the latest version.
</Note>

## Community and support

For bug reports or feature requests, open an issue on the [GitHub repository](https://github.com/top-gg/go-sdk). For general API questions, join the Top.gg Discord server at [discord.gg/dbl](https://discord.gg/dbl).
