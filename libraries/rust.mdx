The Top.gg Rust SDK is a community-maintained crate that wraps the Top.gg REST API for Rust applications. It is not maintained by the Top.gg core team. The GitHub repository is the authoritative source for the crate name, `Cargo.toml` dependency, and usage examples.

## Getting started

**[github.com/top-gg/rust-sdk](https://github.com/top-gg/rust-sdk)**

<Note>
  The repository README contains the `Cargo.toml` dependency snippet and async usage examples. Check there for the latest version published to [crates.io](https://crates.io).
</Note>

## Community and support

For bug reports or feature requests, open an issue on the [GitHub repository](https://github.com/top-gg/rust-sdk). For general API questions, join the Top.gg Discord server at [discord.gg/dbl](https://discord.gg/dbl).
