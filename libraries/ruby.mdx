The Top.gg Ruby SDK is a community-maintained gem that wraps the Top.gg REST API for Ruby applications. It is not maintained by the Top.gg core team. The GitHub repository is the authoritative source for the gem name, `Gemfile` entry, and usage documentation.

## Getting started

**[github.com/top-gg/ruby-sdk](https://github.com/top-gg/ruby-sdk)**

<Note>
  The repository README contains the `gem install` command or `Gemfile` snippet and usage examples. Check there for the latest published version on [RubyGems.org](https://rubygems.org).
</Note>

## Community and support

For bug reports or feature requests, open an issue on the [GitHub repository](https://github.com/top-gg/ruby-sdk). For general API questions, join the Top.gg Discord server at [discord.gg/dbl](https://discord.gg/dbl).
