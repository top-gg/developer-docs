The Top.gg Java SDK is a community-maintained library that wraps the Top.gg REST API for Java applications. It supports posting your bot's server count and retrieving bot and user information. Because it is community-maintained, the GitHub repository is the authoritative source for installation instructions, dependency coordinates, and usage examples.

## Getting started

Visit the GitHub repository for the latest release, Maven/Gradle dependency coordinates, and full usage documentation:

**[github.com/top-gg/java-sdk](https://github.com/top-gg/java-sdk)**

<Note>
  Installation instructions, versioning, and code examples are maintained in the repository's README. Check there for the most up-to-date information.
</Note>

## What the library supports

- Posting your bot's server count to Top.gg
- Retrieving bot information and server counts
- Fetching Top.gg user profiles

## Community and support

This library is not maintained by the Top.gg core team. For bug reports or feature requests, open an issue on the [GitHub repository](https://github.com/top-gg/java-sdk). For general API questions, join the Top.gg Discord server at [discord.gg/dbl](https://discord.gg/dbl).
