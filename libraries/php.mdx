The Top.gg PHP SDK is a community-maintained library for interacting with the Top.gg REST API from PHP. It is not maintained by the Top.gg core team. The GitHub repository is the authoritative source for Composer installation instructions and usage examples.

## Getting started

**[github.com/top-gg/php-sdk](https://github.com/top-gg/php-sdk)**

<Note>
  The repository README contains the Composer require command and code examples. Always check there for the latest release and version constraints.
</Note>

## Community and support

For bug reports or feature requests, open an issue on the [GitHub repository](https://github.com/top-gg/php-sdk). For general API questions, join the Top.gg Discord server at [discord.gg/dbl](https://discord.gg/dbl).
